# 석주잉 게시판

#### Web Board with Spring boot and JPA
- 게시판 기본 서비스 제공
  - 회원 글 / 댓글 / 대댓글 작성, 글 추천(좋아요) 가능
  - 카테고리(말머리, 게시물 개수)별 목록보기, 페이징 등 
- 회원 서비스 제공
  - 회원가입, 로그인, 프로필 사진 등록, 마이페이지 등
- 관리자 서비스 제공
  - 말머리 추가, 공지 작성(상단 노출), 회원의 글 / 댓글 삭제 등
